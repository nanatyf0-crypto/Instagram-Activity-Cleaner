# 🧹 Instagram Auto Unlike Script

سكريبت تلقائي سريع جداً لحذف الإعجابات من Instagram بدون تفاعل يدوي

## ✨ المميزات

- ✅ **تلقائي بنسبة 100%** - لا يحتاج تفاعل يدوي بعد البدء
- ⚡ **سريع جداً** - يعمل في دقائق معدودة
- 🔒 **آمن** - لا يخزن كلمات المرور
- 🎯 **دقيق** - يضغط على كل زر بنفسه
- 📊 **تقارير** - يعرض ما تم إنجازه

## 🚀 البدء السريع

### Windows

```bash
# 1. تثبيت المتطلبات
install.bat

# 2. تشغيل السكريبت
python auto_unlike.py
```

### macOS / Linux

```bash
# 1. تثبيت المتطلبات
bash install.sh

# 2. تشغيل السكريبت
python3 auto_unlike.py
```

## 📋 المتطلبات

- **Python 3.8+**
- **Google Chrome** أو **Chromium**
- **pip** (مدير الحزم)

## 🔧 التثبيت اليدوي

```bash
# 1. تثبيت المتطلبات
pip install -r script_requirements.txt

# 2. تشغيل السكريبت
python auto_unlike.py
```

## 📖 كيفية الاستخدام

1. **قم بتشغيل السكريبت**
   ```bash
   python auto_unlike.py
   ```

2. **أدخل بيانات تسجيل الدخول**
   - اسم المستخدم
   - كلمة المرور

3. **اجلس واستمتع** ☕
   - السكريبت سيقوم بكل شيء تلقائياً:
     - ✅ تسجيل دخول
     - ✅ فتح صفحة النشاط
     - ✅ اختيار الفلتر
     - ✅ تحديد "Oldest First"
     - ✅ تحديد 100 عنصر
     - ✅ الضغط على "Unlike"
     - ✅ تأكيد الإجراء

## ⚙️ خطوات العمل

```
1. تسجيل الدخول 🔐
        ↓
2. الذهاب لصفحة النشاط 📊
        ↓
3. الضغط على الفلتر 🔍
        ↓
4. اختيار "Oldest First" 📅
        ↓
5. اختيار فلتر الإعجابات ❤️
        ↓
6. تحديد 100 عنصر 📦
        ↓
7. الضغط على "Unlike" ❌
        ↓
8. تأكيد الإجراء ✅
```

## 🛡️ الأمان

- ✅ لا نخزن كلمات المرور
- ✅ الاتصال المباشر مع Instagram فقط
- ✅ بدون VPN أو وكيل
- ✅ استخدام Selenium الموثوق

## ⚠️ تحذيرات مهمة

1. **استخدم حسابك الشخصي فقط**
2. **قد يؤدي الاستخدام المكثف لحظر الحساب**
3. **استخدم الفترات الفاصلة بين التشغيل**
4. **لا تشغل السكريبت على عدة حسابات في نفس الوقت**

## 🐛 استكشاف الأخطاء

### لم يتم تسجيل الدخول
- تأكد من صحة اسم المستخدم وكلمة المرور
- تأكد من اتصالك بالإنترنت

### لم يتم العثور على الأزرار
- قد يكون حسابك محدوداً
- قد تكون الصفحة بلغة مختلفة
- حاول تحديث المتصفح

### يتحرك السكريبت ببطء
- هذا طبيعي - Instagram تحتاج وقت للتحميل
- لا تغلق المتصفح أثناء التشغيل
